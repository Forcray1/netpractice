## Description
**NetPractice** is a practical exercise designed to introduce the foundational concepts of computer networking. The primary goal is to configure small-scale simulated networks to make them function correctly by reaching specific connectivity objectives. This project covers essential skills such as IP addressing, managing subnet masks, and understanding the role of hierarchical routing within a network.

## Instructions

### Running the Training Interface
To launch the networking simulation environment:
1. Download the project files from the project page and extract them.
2. Execute the `run.sh` script in your terminal to launch the local web server and open the interface in your browser.
3. If the script fails, manually start a server using: `python3 -m http.server 49242` and navigate to `http://localhost:49242` in your web browser.

### Configuration and Export
*   **Practice:** Enter your login in the designated field to use your personal configuration.
*   **Solving:** Modify the unshaded fields in the network diagrams until the objectives are met.
*   **Exporting:** Once a level is successfully validated, use the **[Get my config]** button to download the configuration file.

### Submission Details
*   You must submit exactly **10 exported configuration files** (one for each completed level).
*   All 10 files must be placed at the **root** of your Git repository.

### Studied Networking Concepts
The completion of this project required a thorough understanding of the following TCP/IP concepts:
*   **TCP/IP Addressing:** Assigning unique identifiers to devices.
*   **Subnet Masks:** Defining network boundaries and host ranges.
*   **Default Gateways:** Configuring the exit point for traffic leaving a local subnet.
*   **Routers and Switches:** Understanding how traffic is directed between networks and handled within local segments.
*   **OSI Layers:** Familiarization with the conceptual layers of network communication.
